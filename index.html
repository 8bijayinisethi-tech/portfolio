Professional Summary

In recent years, moved into AI/LLM engineering — building production agentic systems using LangGraph, RAG pipelines, pgvector, and Snowflake Cortex AI. Comfortable wiring vector databases for semantic retrieval, tuning Redis clusters for caching, and using Celery for async workloads — with full observability through Grafana, Kibana, Prometheus and AWS CloudWatch.

Infrastructure provisioned with Terraform, pipelines automated via Jenkins CI/CD, APIs secured with JWT, SSO, OAuth2 — and everything tested with Pytest and TDD from day one. Three-time award winner. Ships fast. Breaks nothing.
